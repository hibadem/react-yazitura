# Kodluyoruz React Ödev-2
## Installation

Öncelikle projeyi clonelayın.
```bash
git clone https://github.com/hibadem/react-odevler.git
```

## Usage

Projeyi cloneladıktan sonra açınız.

```linux
cd react-odevler/odev-2
```

Node paketlerini yükleyin

```linux
npm install
```

Projeyi çalıştırın

```linux
npm run start
```
